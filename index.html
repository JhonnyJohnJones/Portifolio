Biometria

Este código Python simula um sistema de autenticação biométrica em um site, permitindo a validação de dados como impressões digitais de maneira básica e segura. Ele utiliza algoritmos simples para comparar os dados capturados com informações armazenadas localmente, sem integração com sistemas reais, sendo ideal para fins educacionais ou prototipagem.

Jogo da memória

Este código em Java implementa um jogo da memória com temática de peixes, onde o jogador deve encontrar pares de cartas de espécies marinhas. Utilizando a biblioteca JavaFX, o jogo inclui embaralhamento das cartas, contagem de movimentos, cronômetro e feedback visual. Com um ambiente amigável, o jogo oferece uma experiência divertida e educativa, com opções de ajuste de dificuldade.

Fórum

Este código simula uma postagem em fórum com funcionalidades interativas, permitindo que usuários curtam comentários e vejam o número de curtidas em tempo real. A interface inclui abas com memes aleatórios para tornar a experiência mais divertida. Implementado com HTML, CSS e JavaScript, o código utiliza armazenamento local para gerenciar dados, sem a necessidade de backend, oferecendo uma simulação leve para explorar conceitos de interação e design.

Relógio 25+5

Este site é um relógio de treino interativo apenas no front-end, permitindo ajustar os tempos de treino e descanso. Com uma interface simples e intuitiva, ele exibe um cronômetro e contagem regressiva visual, alertando o usuário quando é hora de mudar de atividade. O site oferece flexibilidade e praticidade para otimizar o desempenho durante os exercícios e pausas, sem necessidade de backend.
